# Movie Collector

Ein modernes Web-Anwendung zum Sammeln und Verwalten von Filmen.

## 🚀 Features

-   Film-Sammlung verwalten
-   Benutzerfreundliche Oberfläche
-   Responsive Design
-   Sichere Benutzerauthentifizierung
-   RESTful API

## 🛠️ Technologien

### Frontend

-   React
-   TypeScript
-   Tailwind CSS
-   Vite

### Backend

-   Go 1.23
-   RESTful API
-   PostgreSQL
-   JWT Authentication

## 📋 Voraussetzungen

-   Go 1.23
-   Node.js 20
-   PostgreSQL
-   Docker (optional)

## 🚀 Installation

1. Repository klonen:

```bash
git clone https://github.com/yourusername/movie-collector.git
cd movie-collector
```

2. Backend einrichten:

```bash
cd backend
go mod download
```

3. Frontend einrichten:

```bash
cd frontend
npm install
```

4. Umgebungsvariablen konfigurieren:
   Kopieren Sie die `.env.example` Datei zu `.env` und passen Sie die Werte an.

## 🏃‍♂️ Entwicklung

### Backend starten

```bash
cd backend
go run main.go
```

### Frontend starten

```bash
cd frontend
npm run dev
```

### Mit Docker starten

```bash
docker-compose up
```

## 🧪 Tests
